## Project: Predict Health Insurance Costs

### Objective
The goal of this project is to predict health insurance costs for individuals based on several input variables.

### Inputs
- Age
- Sex
- BMI (Body Mass Index)
- Number of Children
- Smoker or Non-Smoker
- Region of the Participants
